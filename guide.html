GitHub Beginner Guide

Step by Step Tutorial

GitHub सीखें आसानी से!

शुरुआती लोगों के लिए पूरी गाइड हिंदी में

नीचे दिए गए steps को follow करके GitHub का उपयोग करना सीखें

शुरुआती Steps (Beginner Tasks)

1

GitHub Account बनाएं

सबसे पहले GitHub पर account create करें:

  • github.com पर जाएं
  • "Sign up" button पर click करें
  • Email address, password और username डालें
  • Email verify करें
  • Basic setup complete करें
2

नया Repository बनाएं

Repository (repo) project का folder होता है:

  • Dashboard पर "+" icon पर click करें
  • "New repository" select करें
  • Repository name डालें (जैसे: my-first-project)
  • Description डालें (optional)
  • Public या Private select करें
  • "Create repository" button press करें
3

पहला File Create करें

अपने repository में पहला file add करें:

  • Repository page पर जाएं
  • "Add file" button पर click करें
  • "Create new file" select करें
  • File name डालें (जैसे: README.md)
  • Content लिखें
  • नीचे "Commit changes" section में description डालें
  • "Commit new file" button press करें
4

File Upload करें

Computer से files upload करें:

  • Repository page पर जाएं
  • "Add file" button पर click करें
  • "Upload files" select करें
  • Files select करें या drag & drop करें
  • Commit message डालें
  • "Commit changes" button press करें
5

File Edit करें

Existing files को edit करें:

  • File पर click करें
  • 右上 corner में "Edit" icon (pencil) पर click करें
  • Content modify करें
  • Commit message डालें (जैसे: "fixed typo")
  • "Commit changes" button press करें
6

README.md File बनाएं

अपने project का description लिखें:

  • README.md special file होती है
  • यह repository के front page पर show होती है
  • Markdown format use करें
  • Project के बारे में information डालें
  • Installation instructions लिखें
  • Usage examples दें
# My First Project
This is my first GitHub repository.
## Installation
Clone this repository and run the code.
